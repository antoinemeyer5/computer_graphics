# `georust`

Provides types and methods for geometric concepts (point, line, triangle).

- `Point`
    - Is a `Linvec2` from `linrust`.
- `Line`
    - Is a `Linvec4` from `linrust`.
    - Can get `start` which is a `Linvec2` from `linrust`.
    - Can get `end` which is a `Linvec2` from `linrust`.
- `Triangle`
    - Is 3 `Point`.
